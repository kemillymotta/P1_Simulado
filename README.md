

# Projeto de Avaliaçăo

Este repositório contém a implementaçăo de duas questőes envolvendo listas ligadas:

- **Questăo 1**: Buscar Elemento
- **Questăo 2**: Exibir Menor Valor

---

## Questăo 1: Buscar Elemento

Implemente a funçăo `buscarElemento()` para encontrar um valor em uma lista ligada (suponha que a lista năo contenha valores duplicados).  
- Se o elemento for encontrado, retorne ou exiba o valor.  
- Caso contrário, exiba `"Năo encontrado"`.

**Após testar**, copie e cole a implementaçăo da funçăo no espaço indicado.

---

## Questăo 2: Exibir Menor Valor

Implemente a funçăo `exibirMenorValor()` para percorrer a lista e exibir o menor valor armazenado nos nós.  
Siga estas etapas:  
1. Armazene o valor do primeiro nó em uma variável chamada, por exemplo, `menor`.  
2. Percorra os nós restantes comparando seus valores com `menor`.  
3. Se algum valor for menor, atualize a variável `menor`.  
4. Ao final, exiba o valor de `menor`.
